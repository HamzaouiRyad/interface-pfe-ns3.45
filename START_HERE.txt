🚀 INTEGRATION WITH NS3 (OPTIONAL):

   Want to use real NS3 data instead of mock?

   1. Set up WebSocket server
      → See NS3_INTEGRATION_GUIDE.md (Python/Node.js examples)

   2. Update configuration:
      Create .env.local with:
      NEXT_PUBLIC_WS_URL=ws://your-server:8080/ws

   3. Toggle mode in dashboard
      → Switch from "Mock" to "Real (NS3)"

   4. Live data starts flowing! 🎯
